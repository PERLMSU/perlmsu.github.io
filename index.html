PERL@MSU Software Projects

Here you will find a few software projects that support the work we do in the Physics Education Research Lab at Michigan State University. All work is Creative Commons Attribution-Non-Commerical Licensed .

PhysUtil

PhysUtil is an open-source module that can generate motion diagrams (stroboscopic images), graphs, and timers from within a VPython or Glowscript programs

Pathways

Pathways is a set of python notebooks for analyzing registration data and determining the paths students take through the college towards their degree.

ComputationSurveyAnalysis

ComputationSurveyAnaylsis is a set of python notebooks developed for analyzing a national survey of phyiscs faculty using the Random Forest algorithm.

WebCAT

WebCAT is a web-based formative assessment tool developed to provide students feedback on their development in courses that make use of group work.
